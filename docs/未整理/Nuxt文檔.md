## Nuxt的亮點

### Automation and Convention

Nuxt使用固定的目錄結構完成自動的任務，開發人員只需要專注於功能開發。也可透過配置檔案進行客製化。

- File-based Routing
- Code Splitting `自動分割chunk`
- SSR
- Auto-imports
- Data-fetching utilities
- Zero-config TypeScript: `tsconfig`
- Configured build tools `Vite`
